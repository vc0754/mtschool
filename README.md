# 脉享校园系统手机 APP - 家长版

该套文件使用 Vue Cli 3 工程创建，配合 Mock 数据模拟进行展示，使用到 Vue-Router，Vuex，Axios 等一系列组件。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
